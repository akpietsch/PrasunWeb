Text 3 Preislied auf einen Verdienstfestgeber

  1. In die Almhütte hochgestiegen seiend, hat ein Mann sieben Kühe.
  2. Rechts (in Querrichtung des Tales, talaufwärts blickend) vonǝḍu lošẽ[1] steht eine große Eiche.
  3. An der Stelle baute er eine Almhütte.
  4. Jener Mann heißt Blinder-Dū̃gˈū.
  5. Jeden Tag füllt er ein Maß (mit Milch) und stellt es talauf hin.
  6. Er lud alle sechs Dörfer von Prasun ein. Der Herbst kam.
  7. Er veranstaltete ein (bestimmtes) Verdienstfest.
  8. (Er setzte sich auf den Stuhl) sidĩ und band das rote Band sidĩ (um die Gamaschen)[2].
  9. Sein Name wurde groß.
